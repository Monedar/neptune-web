### Examples of content inside component

`<Alert />` component accepts anything that can be rendered: numbers, strings, elements or an array (or [fragment](https://reactjs.org/docs/fragments.html)) containing these types.

Example of message with bold text and link:

<div className="row">
  <div className="col-md-8">
    <Alert size="sm">
      <b>Lorem</b> ipsum dolor sit amet, consectetur <a href="https://transferwise.com/help">adipisicing elit</a>.
    </Alert>
  </div>
</div>

Example of message with list of items:

<div className="row">
  <div className="col-md-8">
    <Alert size="sm" type="error" arrow="up-center">
      <div>Number is required</div>
      <div>Must be 10 or greater</div>
      <div>Must be 99 or less</div>
    </Alert>
  </div>
</div>
