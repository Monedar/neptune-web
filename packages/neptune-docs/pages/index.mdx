# Neptune Design System

Design guidelines for building interfaces with TransferWise's Design System

**⚠️ This site is a work in progress!**

## Getting started

Explore the site or start by the libraries .

- **[Neptune Components](https://github.com/transferwise/components)** React.js component library for Neptune
- **[Neptune CSS](https://github.com/transferwise/neptune-css)** CSS library for Neptune

## Installation

<Code lang="shell">{`npm i -S @transferwise/components
npm i -S transferwise/neptune-css`}</Code>

## Usage

<Code lang="javascript">{`import "@transferwise/components";
import "~neptune-css/build/css/neptune.css";`}</Code>
